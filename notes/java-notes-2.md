# Estruturas de repetição

```
1. inicialização
2. verificação
3. atualização
```

## Keyword `break`

A palavra interrompe o laço no ponto em que foi chamada.

## Keyword `continue`

A palavra pula o laço atual, sem sair do loop, e ignorando o código restante deste laço.

## Estruturas `while`

Basicamente um `if` que dura.

```java
while(/*verificação*/) {

}
```
